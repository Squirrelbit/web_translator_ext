# web_translator_ext
Started Feb 2025 for a day-long hackathon

MUCH CREDIT GOES TO WORDREFERENCE.COM FOR PROVIDING THE BACKEND.

If you would like to use, download the folder, go to chrome://extensions/ (in chrome), turn on Developer Mode, click Load Unpacked, and open the extension folder.

Written with the help of chrome web documentation and MDN. 
I did not use AI to write the majority of my code because I wanted to learn how to write JS, not how to use an AI copilot.
